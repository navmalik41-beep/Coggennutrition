# Coggen Nutrition - Premium E-Commerce Website

A premium, Apple-style sports nutrition e-commerce platform built with React, Vite, Tailwind CSS, Framer Motion, and Node.js.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, React Router v6, Lucide React
- **Backend**: Node.js, Express, Local JSON Storage (`products.json`)
- **Features**: Hybrid WhatsApp Cart Checkout, Full CRUD Admin Panel, Dynamic Filtering, Premium Animations

## Requirements
- Node.js (v16+ recommended)

## How to run Backend
1. Open a terminal.
2. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   node server.js
   ```
   - The backend will run on `http://localhost:5000`

## How to run Frontend
1. Open a new terminal.
2. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the Vite development server:
   ```bash
   npm run dev
   ```
   - The frontend will be available at `http://localhost:5173` (or as noted in your terminal). Open this URL in your browser.

*(Note: Ensure the backend is running simultaneously so the frontend can fetch product data.)*
